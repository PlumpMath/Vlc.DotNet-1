Vlc.DotNet

Did you ever need a powerfull, yet simple to use audio/video control in your .NET project? Don't want to be forced to jump through loops and hoops to use C++ libraries? Don't want to mess with the WMP COM component? We've got you covered!

More

The power of VLC

You get the total extent of VLC's power and extensibility, along with all of its features. VideoLan DotNet provides all of VLCs basic functionalities and the ability to use any plugin available for VLC. Along with these awesome features you also get the ability to use any command-line switch that you would normally use.

Next

In the comfort of .NET

All of the features come packed inside the .NET framework as an elegant and intuitive control. You'll be able to use the power of VLC regardless of the type of project you're working on because the VLC player control is available for WPF, Windows Forms and Silverlight projects. There's also the option of accessing the introp services directly if want to. All of this while providing you with a easy to use and intuitive debugging interface.

Cool! How do I get started with it?

You can get started right away! Download the binaries, or compile them yourself. After you've downloaded them see how to use them in your project on our wiki. If you still have some problems or questions feel free to contact us.
